# newslab backend

made by Cho jae ho @ 2018


## start
 1. npm install
 2. make config.json file in config/

~~~
### config.json ###

{
  "production": {
    "username" : "root",
    "password" : "yourDB password",
    "database" : "yout database",
    "host" : "localhost",
    "dialect" : "mysql"
  }
}
~~~

## run
> cd bin

> sudo node www
